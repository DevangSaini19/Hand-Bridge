## рҹҺҜ SPECIFIC FIXES

### **Problem: Thumbs Up showing as "Gun"**
**Fix:**
- вң… Close ALL four fingers completely
- вң… Only thumb visible and pointing UP
- вң… Make a tight fist with other fingers
- вң… Hold for 2+ seconds

### **Problem: Number 2 showing as "Peace"**
**Fix:**
- вң… Keep fingers VERY close (touching)
- вң… For "Peace", spread 3-5cm apart
- вң… Check palm facing forward

### **Problem: Getting wrong numbers**
**Fix:**
- вң… Count your extended fingers
- вң… Make sure others are fully curled
- вң… Palm facing forward
- вң… Hold steady 2+ seconds

## рҹ”§ Quick Troubleshooting

**If nothing is recognized:**
1. Check camera is ON (green dot)
2. Click "Start Translation" (should say "Live")
3. Enable subtitles
4. Try showing "5" (easiest gesture)

**If wrong gestures detected:**
1. Improve lighting
2. Hold perfectly still
3. Make gestures more obvious/exaggerated
4. Wait longer (3-4 seconds)
5. Check finger positions match guide

**If voice keeps repeating:**
1. Should be fixed in v2.1
2. If still happens, refresh page
3. Wait full 2 seconds between gestures
